# 🤖 Aplicações Python — Machine Learning

> Trabalho prático sobre aplicações de Machine Learning utilizando Python e Jupyter Notebook.

---

## 📋 Sobre o Projeto

Este repositório contém o desenvolvimento do **Trabalho 4**, com foco em **aplicações de Machine Learning** utilizando Python. O notebook aborda conceitos fundamentais de aprendizado de máquina, desde o pré-processamento de dados até a criação e avaliação de modelos preditivos.

---

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- **Jupyter Notebook**
- **Scikit-learn** — algoritmos de Machine Learning
- **Pandas** — manipulação e análise de dados
- **NumPy** — operações numéricas e matriciais
- **Matplotlib / Seaborn** — visualização de resultados

## 🚀 Como Executar

### Pré-requisitos

Instale as dependências necessárias via pip:

```bash
pip install jupyter scikit-learn pandas numpy matplotlib seaborn
```

### Passos

1. Clone o repositório:
```bash
git clone https://github.com/4L3M40/Trabalho-aplicacoes-python-machine-learning.git
```

2. Acesse a pasta do projeto:
```bash
cd Trabalho-aplicacoes-python-machine-learning
```

3. Inicie o Jupyter Notebook:
```bash
jupyter notebook
```

4. Abra o arquivo `TRABALHO_4.ipynb` no navegador e execute as células sequencialmente.
